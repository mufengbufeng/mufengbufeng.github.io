Unity优化—代码篇—算法数据结构

本文总结一些游戏性能优化，算法和数据结构相关的常见问题。

频繁查找

  • 代码中频繁查找数据结构中的数据

    可以考虑使用（HashSet，Dictionary）等查找效率更高的数据结构

范围查询

  • 代码中频繁进行范围查询

    可以考虑使用空间划分数据结构（四叉树，八叉树，KD树等）来游戏性能优化范围查询效率

过多排序

  • 代码中频繁对大量数据进行排序

    可以考虑使用更高效的排序算法，或者在数据插入时保持数据有序，减少排序次数

大量遍历

  • 代码中频繁遍历大量数据

    1. 可以考虑使用索引结构（如B树，跳表等）来加速数据访问，减少遍历开销
